Alpha-Beta Pruning Explanation

Alpha-Beta pruning is an optimization technique for the minimax algorithm that reduces the number of nodes evaluated in the search tree.

  • Alpha: Best value for the maximizer found so far
  • Beta: Best value for the minimizer found so far
  • Pruning: Skip evaluating moves when we've found evidence that they're worse than a previously examined move

In this 8-puzzle implementation:

  • We use Manhattan distance as the heuristic
  • The goal is to minimize the number of moves to reach the target state
  • Green nodes are being explored
  • Red nodes are pruned
  • Blue nodes are part of the optimal solution path
